# GuessYourSimpson
Expert system of The Simpsons Characters without Expert System libraries (Like experta). Node.js, Svelte, SQL

BackEnd: Api REST in node JS with sequelize.

FrontEnd: Svelte vanilla only web Version.

Database: MySQL.
